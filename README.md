Developed a secure, role-based Military Asset Management System (MAMS) to optimize logistics operations and asset accountability across multiple military bases. The system enables real-time tracking, assignment, transfer, and expenditure of critical defense resources such as vehicles, weapons, and ammunition. It features a responsive React.js frontend with Tailwind CSS, powered by a Node.js (TypeScript, Express) backend and PostgreSQL database. RESTful APIs secured with JWT authentication and Role-Based Access Control (RBAC) ensure data integrity and controlled access across user roles. Prisma ORM streamlines database operations, while Docker and AWS support scalable deployment. The solution delivers transparency, auditability, and efficiency in managing military assets across command hierarchies.
